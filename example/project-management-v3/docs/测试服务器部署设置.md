# 测试服务器部署设置文档

## 一、环境说明

| 环境 | 分支 | 说明 |
|------|------|------|
| 测试服务器 | `staging` | 用于功能测试 |
| 业务服务器 | `main` | 生产环境（暂未部署） |

---

## 二、测试服务器设置步骤

### 步骤 1：安装 Node.js

确保服务器已安装 Node.js 18+：

```bash
node -v
npm -v
```

如未安装，请先安装 Node.js。

### 步骤 2：克隆项目

```bash
# 进入你的项目目录（根据实际情况修改路径）
cd /home

# 克隆项目
git clone https://github.com/fagnxia113/project-management-v3.git

# 进入项目目录
cd project-management-v3
```

### 步骤 3：切换到 staging 分支

```bash
# 拉取所有分支
git fetch origin

# 切换到 staging 分支
git checkout staging

# 拉取最新代码
git pull origin staging
```

### 步骤 4：安装依赖并构建

```bash
# 安装依赖
npm install

# 构建前端
npm run build
```

### 步骤 5：配置自动拉取脚本

创建自动部署脚本：

```bash
# 创建脚本文件
nano /home/auto-deploy-staging.sh
```

写入以下内容：

```bash
#!/bin/bash

# 项目路径（根据实际情况修改）
PROJECT_DIR="/home/project-management-v3"
LOG_FILE="/var/log/auto-deploy-staging.log"

echo "========================================" >> $LOG_FILE
echo "开始自动部署: $(date)" >> $LOG_FILE

cd $PROJECT_DIR

# 拉取最新代码
git fetch origin
LOCAL=$(git rev-parse HEAD)
REMOTE=$(git rev-parse origin/staging)

if [ $LOCAL != $REMOTE ]; then
    echo "检测到新代码，开始更新..." >> $LOG_FILE
    
    # 重置到最新版本
    git reset --hard origin/staging
    
    # 安装依赖
    npm install
    
    # 构建
    npm run build
    
    # 重启服务（如果有 PM2）
    # pm2 restart all
    
    echo "部署完成: $(date)" >> $LOG_FILE
else
    echo "没有新代码更新" >> $LOG_FILE
fi
```

设置脚本权限：

```bash
chmod +x /home/auto-deploy-staging.sh
```

### 步骤 6：配置定时任务

```bash
# 编辑 crontab
crontab -e
```

添加以下内容（每分钟检查一次）：

```bash
*/1 * * * * /home/auto-deploy-staging.sh >> /var/log/auto-deploy-staging.log 2>&1
```

### 步骤 7：创建日志目录

```bash
# 创建日志文件
sudo touch /var/log/auto-deploy-staging.log
sudo chmod 666 /var/log/auto-deploy-staging.log
```

## 六、注意事项

1. **路径修改**：请将 `/home/project-management-v3` 修改为实际的项目路径

2. **数据库配置**：确保 `.env` 文件中的数据库连接配置正确

3. **端口配置**：确保服务器防火墙开放了必要的端口（如 3000、5173 等）

4. **首次部署后**：以后只需要在本地执行 `git push origin staging`，测试服务器会自动更新

## 七、问题排查

### 问题1：权限不足

```bash
sudo chown -R $USER:$USER /home/project-management-v3
```

### 问题2：Git 拉取失败

```bash
# 检查 Git 配置
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 问题3：npm install 失败

```bash
# 清除缓存重试
npm cache clean --force
rm -rf node_modules
npm install
```
